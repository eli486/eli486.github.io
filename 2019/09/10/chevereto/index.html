VM虚拟机自建服务器使用宝塔面板搭建chevereto图床

GJJ · 2019-9-10 · 次阅读

记录VM虚拟机搭建Chevereto图床

Xshell连接linux虚拟机

1.Cnetos安装宝塔面板:

yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && sh install.sh

安装完成，显示如下：

登录宝塔面板

然后，用内网ip替换图中的ip来访问宝塔面板
浏览器输入192.168.159.130:8888/89dc0f29
输入账号密码登录

添加站点


我是在100端口下创建站点，要注意防火墙端口的开放，数据库可以自己修改账号密码。

成功之后的页面（我这ip不一样是之前创建的时候忘了截图，不要在意这些）：

上传目录

点击宝塔面板菜单栏–>文件–>进入站点目录–>上传

将下载好的免费的Chevereto解压的文件上传

上传成功后进入chevereto文件目录，将里面的文件剪切到上级目录

可以删除剪切后的空目录

然后刷新刚才创建成功的页面

上传文件

站点目录app下缺少settings.php文件,创建一个settings.txt,随便输入内容,如123，修改后缀为php

修改权限

站点目录权限不足，需要修改

chattr -i /www/wwwroot/192.168.159.129/.user.ini
chown -R www:www /www/wwwroot/192.168.159.129/

修改完成后再刷新

添加伪静态

宝塔面板菜单栏网站–>点击站点–>弹出页面–伪静态

location / {
            try_files $uri $uri/ /index.php?$query_string;
        }


再刷新页面后

哦嚯！可以安装chevereto了~

安装Chevereto

添加数据库，创建站点时候创的数据库，复制过来

添加邮箱，自己写

完成后

进入admin dashboard

这里有很多配置，设置上传大小、主页等，自己根据需求修改，百度一下就会了

点击上方中间的Chevereto图标进入主页

局域网与外网访问图床

要实现局域网访问，首先要添加端口映射，具体的操作可以参考
VM虚拟机端口映射

外网访问，此处ip就是安装宝塔面板完成后显示的，你也可以通过查询ip地址找到你的ip：
